# Демо-проект тестов для презентации на вакансию тестировщика в Marfatech

Это небольшой проект для демонстрации навыков автоматизации тестирования веб-приложений.  
Проект создан для презентации на вакансию тестировщика и включает основные практики:

- Использование **Page Object Model (POM)** для удобного управления страницами и локаторами.
- Параметризованные тесты для проверки различных сценариев.
- Интеграция с **Allure** для визуального отчёта о результатах тестирования.

---

## Структура проекта

- `pages/` — папка с Page Object для страниц сайта.
- `tests/` — папка с тестами.
- `conftest.py` — фикстуры для Selenium WebDriver и настройки Allure.
- `requirements.txt` — файл с зависимостями проекта.
- `README.md` - краткое описание

---

## Установка зависимостей

Все необходимые пакеты перечислены в файле `requirements.txt`.  
Чтобы установить их, выполните команду:

```bash
pip install -r requirements.txt
```

## Запуск тестов

Для запуска тестов и генерации отчёта Allure выполните:
```bash
pytest --alluredir=reports
```

## Просмотр отчёта:

Для просмотра отчёта выполните:
```bash
allure serve allure-results
```
